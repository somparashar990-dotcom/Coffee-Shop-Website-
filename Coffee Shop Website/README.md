
# Coffee Shop website

A clean, responsive and engaging website designed for a coffee shop, offering customers a smooth experience to explore the menu, learn about the brand and place orders.


## Features

Responsive Design: Adapts perfectly to mobile, tablet, and desktop screens.

Interactive Menu: Browse various coffee blends, seasonal drinks, and new coffee items.

About Us: Displays the story of your roastery, sourcing details, and team.

Order system: Allows customers to easily customize and place orders.

Smooth Animations: Includes scroll reveal animations for a premium user experience.
## Technologies Used

Frontend: HTML5, CSS3, JAVASCRIPT(ES6)

Styling: Tailwind CSS3

Icons/Assets: Fontawesome, Unsplash
